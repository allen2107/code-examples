Февраль-март 2017:

very_first_Spider.py
parcerGUI.py

Парсер данных с использованием феймворка Scrapy для Python 2.7.
В файле very_first_Spider.py описана работа парсера, сначала он авторизируется на сайте, затем переходит на нужную страницу, собирает оттуда данные в файл csv.
Файл parcerGUI.py - GUI программы с использованием Tkinter. Содержит поле для написания целевой ссылки с данными, кнопку вставки url-адреса в поле из буфера обмена, кнопку запуска программы.
